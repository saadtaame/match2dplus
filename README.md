# match2dplus
match2dplus is a generalization of match2d to allow for arbitrary ASCII characters. The program locates occurrences of square matrices in larger square matrices.

Sample input/output:

input:
<pre>
5 2
aACDe
FDhiJ
KlmnO
pqRDh
uvWlm
Dh
lm
</pre>

output
<pre>
match at (1,1)
match at(3,3)
</pre>
